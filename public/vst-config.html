Como configurar o VST

Passos para configurar seu VST

  1. Instale um software VST (utilizamos o Numa Player).
  2. Abra o programa e selecione o(s) instrumento(s) que preferir entre as opções disponíveis.
  3. Conecte seu teclado MIDI ao computador e certifique-se de que está reconhecido pelo sistema pela opção "MIDI Inputs".
  4. Selecione o dispositivo MIDI.
  5. Teste pressionando teclas no teclado MIDI para garantir que o som está saindo pelo VST.
  6. Mantenha o VST aberto enquanto utiliza as atividades musicais do site.

Siga esses passos antes de quaisquer sessão de atividades que vai realizar no site. Aproveite a experiência!
